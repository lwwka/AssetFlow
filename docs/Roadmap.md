# 產品 Roadmap

## 目前階段

`Foundation MVP`

目前的核心任務是把既有前後端骨架，推進成真實資料驅動的 MVP，而不是停留在 mock data 展示階段。

## 近期優先順序

1. 完成 backend auth flow，支援真實 mobile integration
2. 將 Profile 畫面接上 `GET /api/auth/me` (已完成)
3. 補上 portfolio summary endpoint (已完成基礎成本版本)
4. 補上 holdings endpoint (已完成)
5. 補上 transactions endpoint (已完成基礎版本)
6. 讓 dashboard 脫離 mock data

## 里程碑規劃

### Phase 1: Foundation MVP

- 穩定的 login 與 register
- 持續使用者流程
- portfolio creation
- holdings list
- 基本 dashboard summary

目標：

讓使用者可以登入、建立自己的投資組合，並在手機上看到最基本的投資總覽。

### Phase 2: Core Tracking Experience

- transactions
- watchlist persistence
- 更豐富的 portfolio analytics
- 更好的 loading / error states

目標：

讓產品從「可用」提升到「值得日常使用」。

### Phase 3: Automation and Data Integrations

- market data integration
- background sync
- historical performance
- multi-currency support

目標：

降低手動維護資料的成本，提升資料完整性與產品黏著度。

### Phase 4: Intelligence and Personalization

- alerts
- insights
- personalization
- premium reporting

目標：

從單純追蹤工具進一步走向個人化投資助理與進階報表能力。
